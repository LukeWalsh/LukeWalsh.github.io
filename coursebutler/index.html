Course Butler

Summer 2013 - HTML/CSS, Javascript (jQuery), MongoDB

Course Butler is a service I am developing. It will notify users via text message when a seat opens up in a class they have subscriped to.

Course Butler is my first time using meteor. Meteor allows functional reactive programming in a Model View-Model View (MVVM) paradigm. This is incredibly powerful, and I truly believe that this is the direction the web is heading.

The app will be live on coursebutler.com as soon as I am finished developing it.
